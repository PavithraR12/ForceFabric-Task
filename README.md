# Mini Task Manager

This repository contains the **Mini Task Manager** project, organized into three main modules:

- **Frontend**: User interface built with modern web technologies.
- **Backend**: API and server-side logic.
- **Tic-Tac-Toe**: A fun game module integrated into the project.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16+ recommended)
- npm or yarn package manager
- Git

### Installation
Clone the repository:
```bash
git clone https://github.com/PavithraR12/ForceFabric-Task.git
cd mini-task-manager
cd frontend && npm install
cd ../backend && npm install
cd ../tic-tac-toe && npm install
cd frontend
npm run dev
cd backend
npm run start
cd tic-tac-toe
npm run start
